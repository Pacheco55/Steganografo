## **Cómo Funciona**

### **Ocultar Información**

Escribe tu mensaje secreto → Selecciona un método de ocultación → Genera texto de cobertura → Comparte el texto "inocente" → Solo quien conoce el método puede recuperar el mensaje.

### **Recuperar Información**

Pega el texto sospechoso → Presiona "Decodificar Automático" → El mensaje oculto se revela instantáneamente.

### **Analizar Textos**

¿Sospechas que un texto contiene información oculta? El módulo forense detecta anomalías, patrones sospechosos y caracteres invisibles.

**¿Es legal usar esteganografía?**
Sí. Ocultar información no es ilegal. Es una técnica legítima de privacidad usada globalmente.

Los métodos están diseñados para resistir análisis forense básico. La detección profesional requiere herramientas especializadas y conocimiento del método usado.
